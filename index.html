The ITP GitHub Hall of Fame

What?
This is a sorted list of the 50 most popular ITP code repositories on GitHub.
Why does my super popular repository not on the list?
The repositories on the list are such that include the term "itp" on their description.
Add "itp" to your repository description and come back to watch it enters to the hall of fame.
